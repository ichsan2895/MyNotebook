Penting untuk diperhatikan, nama negara pada file JSON harus sama persis dengan nama negara pada dataframe yang ada. Semisal jikalau di JSON = 'United States of America' namun di dataframe kolom "Country" masih "US", maka tidak akan terbaca.

featureidkey adalah konektor antara JSON dengan dataframe tersebut. Dimana nama negara pada JSON terletak pada properties.name (lihat hasil jawaban di atas mengenai name : 'Afganistan')

import plotly.graph_objects as go
from plotly.subplots import make_subplots
import pandas as pd

# Make figure with subplots
fig = make_subplots(rows=1, cols=2, specs=[[{"type": "bar"},
                                            {"type": "surface"}]])

# Add bar traces to subplot (1, 1)
fig.add_trace(go.Bar(y=[2, 1, 3]), row=1, col=1)
fig.add_trace(go.Bar(y=[3, 2, 1]), row=1, col=1)
fig.add_trace(go.Bar(y=[2.5, 2.5, 3.5]), row=1, col=1)
fig.add_trace(go.Bar(y=[4, 2, 3]), row=1, col=1)
fig.add_trace(go.Bar(y=[5, 1, 2]), row=1, col=1)

# Add surface trace to subplot (1, 2)
# Read data from a csv
z_data = pd.read_csv("mt_bruno_elevation.csv")
fig.add_surface(z=z_data)

# Hide legend
fig.update_layout(
    showlegend=False,
    title_text="Version 4 Default Theme",
    height=500,
    width=800,
)

fig.show()
